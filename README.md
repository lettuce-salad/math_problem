# factor_generator.py

## 概要

このプログラムは、整数の素因数分解問題を自動生成し、問題と解答をそれぞれ別ファイルに出力するPythonスクリプトです。

指定した素数の範囲と最大値に基づいてランダムに整数を生成し、その素因数分解結果を整形して出力します。数学の演習問題や小テスト作成に適しています。

---

## 主な機能

* エラトステネスの篩による素数生成
* 指定範囲の素数を用いた整数のランダム生成
* 整数の素因数分解
* 指数を上付き文字で整形した解答表示
* 問題と解答を別ファイルで出力

---

## 動作環境

* Python 3.x
* 標準ライブラリのみ使用（追加インストール不要）

---

## 使い方

### 1. 実行

```bash id="qv8k2d"
python factor_generator.py
```

### 2. 入力項目

実行後、以下を入力します：

* 出題する問題数
* 使用する素数の最大値
* 問題に使用する整数の最大値

例：

```id="g2lm9x"
何問出題しますか？: 10
素因数に使う最大の素数はいくつですか？: 10
問題に使う数の最大値はいくつですか？: 1000
```

---

## 出力ファイル

### questions.txt

```id="m3k1az"
次の数を素因数分解せよ（素因数は最大 10 以下、値は最大 1000）：
1. 360
2. 840
...
```

### answers.txt

```id="s8p4cw"
解答：
1. 2³ × 3² × 5
2. 2³ × 3 × 5 × 7
...
```

---

## プログラム構成

### `sieve(n)`

n以下の素数を列挙する関数（エラトステネスの篩）

### `generate_number(primes, max_value)`

指定された素数から、最大値以下の整数をランダムに生成

### `prime_factors(n)`

整数nを素因数分解し、指数付きで管理（Counter使用）

### `to_superscript(num)`

整数を上付き文字に変換

### `format_factors(factors)`

素因数分解結果を整形（例：2³ × 3 × 5²）

### `main()`

入力処理、問題生成、ファイル出力を統括

---

## 注意点

* 条件によっては適切な数が生成できず、問題数が不足する場合があります
* 同じ問題が重複する可能性があります（ランダム生成のため）
* 最大値が小さすぎると生成に失敗しやすくなります

# root.py

## 概要

このプログラムは、平方根（ルート）の簡単化問題を自動生成し、問題と解答を1つのテキストファイルに出力するPythonスクリプトです。

「√の中を外に出す」タイプの計算練習に特化しており、授業用プリントや演習問題の作成に適しています。

---

## 主な機能

* 完全な平方数（例：√36 = 6）の問題生成
* 平方根の簡単化問題（例：√72 = 6√2）の生成
* 問題の重複防止
* 問題と解答のペアをシャッフル
* 1ファイルに問題と解答をまとめて出力

---

## 動作環境

* Python 3.x
* 標準ライブラリのみ使用（追加インストール不要）

---

## 使い方

### 1. 実行

```bash id="20nkpa"
python root.py
```

### 2. 入力項目

実行後、以下を入力します：

* 作成する問題数
* ルートの中の最大値
* √の中に残る数（inner）の最大値
* 完全な平方数の問題数

---

## 出力ファイル

### roots_problems.txt（デフォルト）

```id="cmwzlt"
【平方根を簡単にする問題（全20問）】
1. √72
2. √36
3. √50
...

【解答】
1. 6√2
2. 6
3. 5√2
...
```

---

## プログラム構成

### `generate_root_simplification_problems(...)`

問題生成とファイル出力を行うメイン関数

#### 引数

* `num_problems`：総問題数
* `max_value`：√の中の最大値
* `inner_max`：√の中に残る数の最大値
* `num_perfect_squares`：完全平方数の問題数
* `filename`：出力ファイル名（デフォルト：roots_problems.txt）

---

## 注意点

* 条件によっては指定した問題数を満たせない場合があります
* 完全平方数の問題数は、総問題数以下にする必要があります

---

## 活用例

* 中学生向け平方根の演習問題作成
* 小テスト・プリント生成

---

# linear_equation.py

## 概要

このプログラムは、一次方程式の問題を自動生成し、問題と解答をLaTeX形式（.texファイル）で出力するPythonスクリプトです。

展開を含む一次方程式（例：( a(bx+c)=dx+e )）を生成し、計算練習やテスト作成に利用できます。

---

## 主な機能

* ランダムな一次方程式の生成
* 展開を含む式（分配法則の練習）
* 解が整数になるように問題を設計
* LaTeX形式で問題・解答を出力
* そのままPDF化可能な構造

---

## 動作環境

* Python 3.x
* LaTeX環境（PDF化する場合）

---

## 使い方

### 1. 実行

```bash id="y8k2dm"
python linear_equation.py
```

### 2. 出力

実行すると、以下のファイルが生成されます：

* `linear_equations.tex`

---

## 出力ファイル

### linear_equations.tex

* 前半：問題
* 後半：解答（改ページ後）

例：

```latex id="k4v9ps"
\section*{一次方程式}

\[1.\; 2(3x+1) = 5x+7\]

\newpage
\section*{解答}

\[1.\; x = 2\]
```

※LaTeXをコンパイルすることでPDFとして利用可能

---

## プログラム構成

### `linexpr(a, b)`

一次式 (ax + b) を文字列として整形
（例：`2x+3`, `-x-5` など）

### `parenexpr(a, b, c)`

括弧付きの式 (a(bx+c)) を生成

### 問題生成ロジック

* 解 (x) を先に決定
* 左辺 (a(bx+c)) を生成
* 右辺 (dx+e) を、解が一致するように計算して構築
  → 必ず整数解を持つ問題になる

---

## 設定項目

```python id="c9r2lm"
n = 15
coef_range = [i for i in range(-9, 10) if i != 0]
```

* `n`：問題数
* `coef_range`：係数の範囲（0は除外）

---

## 注意点

* 係数はランダム生成のため、難易度にばらつきがあります
* LaTeX環境がない場合はPDF化できません
* 分数解は生成されません（すべて整数解）
